Here are some plugins I wrote. Mostly effects. The synths like PAPU, SID, etc. have their own repos.

## Building

Each plugin can be built independently from its own directory:

```bash
cd plugins/<plugin_name>
cmake -B build
cmake --build build --config Release
```

Or build all plugins at once from the root directory:

```bash
cmake -B build
cmake --build build --config Release
```
